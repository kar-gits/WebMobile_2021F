What is HTML?

The HyperText Markup Language, or HTML is the standard markup language for documents
designed to be displayed in a web browser.

  • Web pages development
  • Web document Creation
  • Internet navigation

What is CSS?

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML.

  1. keeps content separate from how to display
  2. can reuse the display styles on different contents without repeating css code
  3. makes it easy to maintain the code
